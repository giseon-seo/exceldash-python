# 📊 Excel Dashboard

Excel 데이터를 업로드하여 다양한 차트와 대시보드를 생성하는 Python 웹 애플리케이션입니다.

## 🚀 주요 기능

- **📊 자동 대시보드 생성**: 업로드된 데이터를 분석하여 관련된 여러 차트를 자동으로 생성
- **📈 다양한 차트 타입**: 막대그래프, 선그래프, 파이차트, 산점도, 히스토그램, 박스플롯, 히트맵, 영역차트
- **📋 데이터 탐색**: 데이터 필터링, 정렬, 다운로드 기능
- **📊 요약 통계**: 기술통계, 분포 분석
- **🔍 데이터 분석**: 결측값 분석, 데이터 타입 정보
- **🔬 고급 데이터 분석**: 통계 검정, 군집 분석, PCA, 시계열 분석, 재무 분석
- **💾 데이터 내보내기**: 차트와 데이터를 다양한 형식으로 다운로드

## 🛠️ 설치 방법

1. **저장소 클론**
```bash
git clone <repository-url>
cd exceldash-python
```

2. **가상환경 생성 및 활성화**
```bash
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate  # Windows
```

3. **의존성 패키지 설치**
```bash
pip install -r requirements.txt
```

## 🚀 실행 방법

### Streamlit 앱 실행
```bash
streamlit run app.py
```
브라우저에서 `http://localhost:8501`로 접속

### Dash 앱 실행
```bash
python dash_app.py
```
브라우저에서 `http://localhost:8050`로 접속

## 📊 지원하는 차트 타입

### 기본 차트
- **막대그래프**: 범주별 수치 비교
- **선그래프**: 시계열 데이터 트렌드
- **파이차트**: 비율 및 구성 분석
- **산점도**: 두 변수 간 관계 분석
- **히스토그램**: 데이터 분포 분석
- **박스플롯**: 분포 및 이상치 탐지
- **히트맵**: 상관관계 분석
- **영역차트**: 누적 트렌드 분석

### 자동 대시보드 차트
- **상관관계 히트맵**: 변수 간 상관관계 시각화
- **데이터 분포 히스토그램**: 수치형 데이터 분포
- **범주별 박스플롯**: 범주형 변수별 수치 분포
- **범주별 막대그래프**: 범주형 데이터 집계
- **비율 파이차트**: 구성 비율 분석
- **산점도**: 변수 간 관계 분석
- **월별/분기별 집계**: 시계열 데이터 집계
- **상위/하위 분석**: 순위별 분석
- **데이터 분포 분석**: 구간별 분류 분석
- **성장률 분석**: 시계열 성장률
- **평균 vs 중앙값**: 중심경향성 비교
- **표준편차 분석**: 변동성 분석
- **범주 조합 분석**: 다중 범주 분석
- **분위수 분석**: 분포 위치 분석
- **이상치 탐지**: 비정상 데이터 식별

## 🔬 고급 데이터 분석 기능

### 📊 기술통계 분석
- 평균, 중앙값, 표준편차, 분산
- 최소값, 최대값, 사분위수
- 왜도, 첨도 (분포 형태 분석)
- 결측값 분석

### 🔗 상관관계 분석
- Pearson 상관계수 계산
- 유의성 검정 (p-value)
- 상관계수 행렬 시각화
- 유의한 상관관계 식별

### ⚠️ 이상치 탐지
- IQR 방법 (Interquartile Range)
- Z-score 방법
- 이상치 비율 및 개수 분석
- 이상치 경계값 계산

### 📈 정규성 검정
- Shapiro-Wilk 검정
- Kolmogorov-Smirnov 검정
- 정규분포 여부 판정
- 검정 통계량 및 p-value

### 📈 시계열 분석
- 선형 회귀 기반 트렌드 분석
- 기울기, R², MSE 계산
- 추세 방향 및 강도 분석
- 월별/분기별 계절성 분석

### 🎯 군집 분석
- K-means 군집화
- 데이터 정규화
- 군집 중심 및 크기 분석
- Inertia (군집 품질 지표)

### 🔍 주성분 분석 (PCA)
- 차원 축소
- 설명된 분산 비율
- 누적 설명 분산
- 주성분 계수 분석

### 💰 재무 분석
- 수익성 지표 (ROE, ROA)
- 마진 분석 (순이익률, 총이익률)
- 안정성 지표 (부채비율)
- 성장성 지표 (매출 성장률)

## 📊 샘플 데이터

프로젝트에는 다음과 같은 샘플 데이터가 포함되어 있습니다:

### 일반 데이터
- **Sales**: 판매 데이터 (제품, 지역, 매출 등)
- **Employees**: 직원 데이터 (부서, 급여, 성과 등)
- **Weather**: 날씨 데이터 (온도, 습도, 강수량 등)
- **Stocks**: 주식 데이터 (가격, 거래량, 변동률 등)

### 회계/재무 데이터 (회계사 지원용)
- **Financial**: 기업별 재무제표 데이터 (매출액, 영업이익, ROE, ROA, 부채비율 등)
- **Sales_Financial**: 부서별/제품별 매출 및 수익성 데이터 (매출액, 매출원가, 이익률 등)
- **Accounting**: 계정과목별 회계 거래 데이터 (매출액, 매출원가, 판매비, 관리비 등)

## 회계 업무 적용 사례

이 프로젝트는 다음과 같은 회계 업무에 직접 적용 가능합니다:

- **재무제표 분석**: 기업별 재무비율 비교 및 트렌드 분석
- **수익성 분석**: 제품별/부서별 수익률 분석 및 개선점 도출
- **비용 분석**: 계정과목별 비용 구조 분석 및 예산 관리
- **이상 거래 탐지**: 비정상적인 거래 패턴 식별
- **예측 분석**: 과거 데이터 기반 미래 성과 예측

## 🛠️ 사용 방법

1. **데이터 업로드**: 사이드바에서 Excel 파일 (.xlsx, .xls) 업로드
2. **시트 선택**: 여러 시트가 있는 경우 원하는 시트 선택
3. **대시보드 확인**: 자동 생성된 대시보드에서 전체적인 데이터 분석
4. **차트 생성**: 원하는 차트 타입을 선택하여 커스텀 차트 생성
5. **데이터 탐색**: 데이터 보기 탭에서 원본 데이터 확인 및 필터링
6. **통계 분석**: 요약 통계 및 데이터 분석 탭에서 기본 통계 확인
7. **고급 분석**: 고급 분석 탭에서 전문적인 통계 분석 수행
